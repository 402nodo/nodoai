## 📈 1. Yield Farming

### Концепция

Покупаем исходы с **очень высокой вероятностью** (95%+), которые торгуются чуть ниже $1.00.

```
┌─────────────────────────────────────────────────────────┐
│  ПРИМЕР                                                  │
│                                                         │
│  Рынок: "Будет ли Рождество 25 декабря 2025?"          │
│  YES = $0.98                                            │
│                                                         │
│  Инвестиция: $100                                       │
│  Покупаем: 102 shares @ $0.98                          │
│  При резолюции: 102 × $1.00 = $102                     │
│  Профит: $2 (2%)                                        │
│                                                         │
│  Если до резолюции 30 дней:                            │
│  APY = (2% / 30) × 365 = 24% годовых                   │
└─────────────────────────────────────────────────────────┘
```

### Критерии отбора

| Параметр | Минимум | Идеально |
|----------|---------|----------|
| Вероятность | 95% | 97%+ |
| Volume | $10,000 | $100,000+ |
| Дней до резолюции | 1 | 7-30 |
| APY | 20% | 50%+ |

### Risk Levels

| Уровень | Вероятность | Риск | APY |
|---------|-------------|------|-----|
| 🟢 SAFE | 97%+ | Очень низкий | 10-30% |
| 🟡 MODERATE | 95-97% | Низкий | 30-50% |
| 🔴 RISKY | 90-95% | Средний | 50-100% |

### AI Analysis для Yield

AI модели проверяют:
1. **Реальность события** — действительно ли вероятность такая высокая?
2. **Скрытые риски** — что может пойти не так?
3. **Timing** — нет ли событий до резолюции?
4. **Liquidity** — можно ли выйти если нужно?

## 🔄 2. Delta Neutral

### Концепция

Находим **логические несоответствия** между связанными рынками.

```
┌─────────────────────────────────────────────────────────┐
│  ПРИМЕР 1: Ценовые пороги                               │
│                                                         │
│  "BTC достигнет $150K" YES = $0.08                     │
│  "BTC достигнет $100K" YES = $0.05  ← ОШИБКА!          │
│                                                         │
│  Логика: Если BTC дойдёт до $150K, он ОБЯЗАТЕЛЬНО      │
│  пройдёт $100K. Значит $100K YES ≥ $150K YES           │
│                                                         │
│  Действие: Покупаем $100K YES @ $0.05                  │
│  Потенциал: минимум $0.08 (60% профит)                 │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  ПРИМЕР 2: Dip события                                  │
│                                                         │
│  "BTC упадёт до $50K" YES = $0.10                      │
│  "BTC упадёт до $70K" YES = $0.08  ← ОШИБКА!           │
│                                                         │
│  Логика: Чтобы упасть до $50K, BTC должен сначала      │
│  упасть до $70K. Значит $70K YES ≥ $50K YES            │
│                                                         │
│  Действие: Покупаем $70K YES @ $0.08                   │
└─────────────────────────────────────────────────────────┘
```

### Типы несоответствий

| Тип | Описание | Пример |
|-----|----------|--------|
| **Threshold** | Разные ценовые уровни | BTC $100K vs $150K |
| **Subset** | Одно включает другое | "Win election" vs "Win state" |
| **Inverse** | Противоположные | YES + NO ≠ 100% |
| **Temporal** | Временные связи | "Q1" vs "2025" |

### AI Analysis для Delta

AI модели проверяют:
1. **Логическая связь** — действительно ли события связаны?
2. **Скрытые различия** — нет ли нюансов в формулировках?
3. **Ликвидность** — можно ли исполнить сделку?
4. **Timing risk** — не закроется ли один рынок раньше?

## 🎲 3. Multi-Outcome Arbitrage

### Концепция

На рынках с 3+ исходами, если сумма цен < 100%, можно купить ВСЕ исходы и гарантировать профит.

```
┌─────────────────────────────────────────────────────────┐
│  ПРИМЕР: "Кто станет спикером?"                         │
│                                                         │
│  Johnson  = $0.45                                       │
│  McCarthy = $0.30                                       │
│  Other    = $0.20                                       │
│  ──────────────────                                     │
│  TOTAL    = $0.95  ← меньше $1.00!                     │
│                                                         │
│  Инвестиция: $95 (купить все исходы)                   │
│  Гарантированный возврат: $100                         │
│  Профит: $5 (5.3%)                                      │
└─────────────────────────────────────────────────────────┘
```

### Когда работает

| Условие | Почему важно |
|---------|--------------|
| Сумма < 97% | Покрывает комиссии |
| High volume | Можно исполнить |
| Clear rules | Один исход точно сработает |

### AI Analysis для Multi-Outcome

AI модели проверяют:
1. **Полнота** — действительно ли один исход обязателен?
2. **Правила резолюции** — нет ли edge cases?
3. **Комиссии** — учтены ли все fees?
4. **Execution** — можно ли купить все исходы по этим ценам?

## 📊 4. Momentum Trading

### Концепция

Отслеживаем резкие изменения цен и входим в тренд раньше толпы.

```
┌─────────────────────────────────────────────────────────┐
│  СИГНАЛ                                                 │
│                                                         │
│  Рынок: "Trump wins 2024"                              │
│  Вчера: $0.45                                          │
│  Сейчас: $0.52  (+15.5% за 24ч)                        │
│                                                         │
│  Volume: $2M → $5M (+150%)                             │
│                                                         │
│  Возможные причины:                                     │
│  • Новый опрос                                         │
│  • Новость                                             │
│  • Whale activity                                       │
│                                                         │
│  AI анализирует: Обоснован ли рост? Продолжится?       │
└─────────────────────────────────────────────────────────┘
```

### Сигналы

| Сигнал | Триггер | Значение |
|--------|---------|----------|
| 🚀 Surge | +10% за 24ч | Сильный интерес |
| 📈 Trend | +5% 3 дня подряд | Устойчивый рост |
| 🐋 Whale | Single trade >$50K | Крупный игрок |
| 📊 Volume | 3x average | Повышенная активность |

### AI Analysis для Momentum

AI модели проверяют:
1. **Причина движения** — новость, опрос, манипуляция?
2. **Фундаментал** — соответствует ли цена реальности?
3. **Sentiment** — что говорят в соцсетях?
4. **Technical** — перекуплено/перепродано?

## ⚠️ Risk Management

| Правило | Описание |
|---------|----------|
| **Диверсификация** | Не более 10% на один рынок |
| **Liquidity check** | Убедись что можешь выйти |
| **Time limit** | Не держи позицию слишком долго |
| **Stop loss** | Если упало на 20% — пересмотри |
| **Take profit** | Фиксируй прибыль частями |
