## 🎯 Почему консенсус лучше?

### Проблема одной модели:

| Проблема | Описание |
|----------|----------|
| **Bias** | Каждая модель имеет свои предубеждения |
| **Галлюцинации** | AI может выдумывать факты |
| **Устаревшие данные** | Training cutoff может быть старым |
| **Слепые зоны** | Некоторые темы модель знает хуже |

### Решение через консенсус:

| Преимущество | Как работает |
|--------------|--------------|
| **Снижение bias** | Разные модели компенсируют друг друга |
| **Детекция ошибок** | Если 5 согласны, а 1 нет — проверяем |
| **Актуальность** | Разные модели с разным cutoff |
| **Полнота** | Объединяем сильные стороны каждой |

## 💰 Монетизация: 402 Protocol

### Что такое 402?

HTTP статус **402 Payment Required** — зарезервирован для микроплатежей.

```
┌─────────────────────────────────────────────────────────┐
│                    REQUEST FLOW                          │
│                                                          │
│  User Request                                            │
│       │                                                  │
│       ▼                                                  │
│  ┌─────────────┐                                        │
│  │ 402 Gateway │ ← Проверка баланса/подписки            │
│  └──────┬──────┘                                        │
│         │                                                │
│    ┌────┴────┐                                          │
│    ▼         ▼                                          │
│  ✅ Paid    ❌ Not Paid                                  │
│    │              │                                      │
│    ▼              ▼                                      │
│  Process      Return 402                                │
│  Request      + Payment Info                            │
│    │                                                     │
│    ▼                                                     │
│  Return Analysis                                        │
└─────────────────────────────────────────────────────────┘
```

## 🔄 Workflow

### 1. Пользователь запрашивает анализ

```json
POST /api/analyze
{
  "market_url": "polymarket.com/event/btc-150k",
  "depth": "deep",  // quick | standard | deep
  "strategy": "yield_farming"
}
```

### 2. Система списывает микроплатёж

```json
// 402 Response (если не оплачено)
{
  "error": "Payment Required",
  "amount": "$0.10",
  "payment_methods": ["solana_usdc", "lightning", "stripe"]
}
```

```

```

## 🎲 Стратегии

NODO поддерживает несколько стратегий анализа:

### 1. 📈 Yield Farming
Поиск "очевидных" событий с высокой вероятностью (95%+)

### 2. 🔄 Delta Neutral  
Поиск логических ошибок между связанными рынками

### 3. 🎯 Multi-Outcome Arbitrage
Арбитраж на рынках с 3+ исходами

### 4. 📊 Momentum
Отслеживание резких движений цен
